# Agentic QA OS

에이전트 기반 QA 워크플로우를 소개하는 React + Vite + Tailwind UI 데모입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```

## GitHub

원격 저장소: [https://github.com/queenlynda1223-max/Agentic-QA-OS](https://github.com/queenlynda1223-max/Agentic-QA-OS)

## 배포 (링크로 미리보기)

[Vercel](https://vercel.com) 또는 [Netlify](https://netlify.com)에서 이 저장소를 연결하면 `https://…` 주소로 공개할 수 있습니다. (Framework preset: Vite, build: `npm run build`, output: `dist`)
